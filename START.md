# Швидкий старт: агент налаштовує все сам

Це найкоротший шлях. Ви встановлюєте Claude Code на свій компʼютер, вставляєте один промпт, і далі він робить все сам: перевіряє акаунти, генерує ключі, підключається до сервера, встановлює і налаштовує кожен модуль. Ви лише оплачуєте підписки, натискаєте кнопки підтвердження і відповідаєте на його питання.

Перед стартом пройдіть [модуль 00](modules/00-accounts/README.md): створіть акаунти й оплатіть підписки. Без акаунтів Claude і ChatGPT промпт не запуститься.

## Крок 1. Встановіть Claude Code

### Windows

1. Відкрийте PowerShell: натисніть Win, наберіть `powershell`, Enter.
2. Вставте команду і натисніть Enter:

   ```powershell
   irm https://claude.ai/install.ps1 | iex
   ```

3. Закрийте і знову відкрийте PowerShell, наберіть:

   ```powershell
   claude
   ```

4. Claude Code відкриє посилання для входу. Оберіть вхід через підписку (Sign in with Claude account) і підтвердьте в браузері.

Якщо команда `claude` не знаходиться після перевідкриття PowerShell, перезавантажте компʼютер один раз.

### macOS

1. Відкрийте Термінал: Cmd+Пробіл, наберіть `Terminal`, Enter.
2. Вставте команду і натисніть Enter:

   ```sh
   curl -fsSL https://claude.ai/install.sh | bash
   ```

3. Наберіть:

   ```sh
   claude
   ```

4. Увійдіть через підписку за посиланням, яке відкриє Claude Code.

## Крок 2. Вставте майстер-промпт

У відкритому Claude Code вставте цілком вміст файлу
[prompts/master-bootstrap.md](prompts/master-bootstrap.md)
(відкрийте лінк, натисніть кнопку копіювання на GitHub, вставте в Claude Code, Enter).

Далі агент веде вас сам, за повним маніфестом інсталяцій у строгому порядку (22 пункти, від пакетного менеджера до фінального аудиту безпеки):

- поставить на ваш компʼютер все, чого бракує: Homebrew і git на Mac, git через winget на Windows, Tailscale, 1Password з CLI;
- пройде з вами чеклист акаунтів і відкриє потрібні сторінки;
- згенерує SSH-ключ і допоможе створити сервер у Hetzner;
- сам захистить сервер, підключить Tailscale, 1Password, домен;
- сам встановить агента на сервер і налаштує Telegram, пошту, нотатки, cron, бекапи;
- покличе вас лише там, де без людини ніяк: оплата, вхід у акаунти, кнопки Allow;
- завершить обовʼязковим аудитом безпеки: жодного відкритого порту, всі секрети в 1Password, всі токени з мінімальними правами, і покаже вам це доказово.

Сетап можна переривати. Напишіть потім у Claude Code «продовжуй налаштування» і він підхопить з того місця, де зупинились: прогрес зберігається у файлі на вашому компʼютері.
