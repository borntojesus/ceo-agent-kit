# 02. Tailscale VPN

Ховаємо сервер від інтернету. Після цього модуля сервер приймає з'єднання тільки з ваших пристроїв, а для сканерів і ботів з інтернету він просто не існує.

## Що ви отримаєте

Приватну мережу (tailnet) між вашими пристроями і сервером. SSH працює тільки через неї. Публічна IP-адреса сервера більше не приймає жодних з'єднань.

## Що зробити руками

1. Створіть акаунт на [tailscale.com](https://tailscale.com) (вхід через Google або GitHub, план Free).
2. Встановіть Tailscale на свій комп'ютер і телефон: [tailscale.com/download](https://tailscale.com/download). Залогіньтесь на обох.
3. Тримайте відкритою [консоль Tailscale](https://login.tailscale.com/admin/machines): за хвилину там з'явиться ваш сервер.

## Що зробить агент

Зайдіть на сервер (`ssh root@ВАШ_IP`), запустіть `claude` і вставте цілком файл
[prompts/join-tailnet.md](prompts/join-tailnet.md). Агент встановить Tailscale на Proxmox-хост, опублікує через нього внутрішню мережу контейнерів (один перемикач у консолі Tailscale), дочекається вашого підтвердження, що все працює, і лише тоді закриє публічний SSH.

## Перевірка

1. У [консолі Tailscale](https://login.tailscale.com/admin/machines) сервер у списку, статус Connected.
2. З ноутбука все відкривається за tailnet-адресами:

   ```sh
   ssh root@agent-host        # хост
   ssh agent@10.10.10.100     # контейнер агента
   ```

   А Proxmox веб-інтерфейс: `https://agent-host:8006` (попередження про сертифікат очікуване).

3. Публічний доступ закритий: команда `ssh agent@ПУБЛІЧНИЙ_IP` більше не з'єднується (висить і відвалюється по таймауту). Це очікувана поведінка.

З цього моменту вся дорога на сервер і контейнери йде тільки через Tailscale.

## Куди зберегти доступи

- "Tailscale account": яким акаунтом залогінені (Google або GitHub).
