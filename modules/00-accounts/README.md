# 00. Акаунти й підписки

Перед стартом потрібні сім акаунтів. Два з них платні підписки на AI, решта дешеві або безкоштовні. Це єдиний модуль, де все робиться руками: далі більшість роботи виконуватиме агент.

## Що ви отримаєте

Всі акаунти, на які спиратиметься агент. Все оформлено на вас: ви власник кожного сервісу, кожного токена і кожного байта своїх даних.

## Що зробити руками

1. **ChatGPT Plus** (20 доларів на місяць): [chatgpt.com](https://chatgpt.com) → Upgrade plan → Plus. Потрібен для Codex CLI, другого "мозку" агента для кодових задач.
2. **Claude Pro** (20 доларів на місяць): [claude.ai](https://claude.ai) → Upgrade → Pro. Це основний двигун агента: Claude Code працює за цією підпискою без окремої оплати за API. Якщо плануєте користуватись агентом інтенсивно щодня, беріть Max (100 доларів): ліміти Pro можуть закінчуватись.
3. **GitHub** (безкоштовно): [github.com/signup](https://github.com/signup). Тут житимуть ваші нотатки (приватне репо) і майбутні проєкти.
4. **Hetzner** (сервер, близько 8 евро на місяць): [console.hetzner.com](https://console.hetzner.com) → Sign up. Німецький хостинг, на якому житиме агент. При реєстрації попросять підтвердити особу.
5. **Cloudflare** (безкоштовно): [dash.cloudflare.com/sign-up](https://dash.cloudflare.com/sign-up). Керуватиме вашим доменом.
6. **1Password** (3 долари на місяць): [1password.com](https://1password.com) → Individual. Сейф для всіх паролів і токенів, які з'являться далі. Встановіть застосунок на комп'ютер і телефон.
7. **Telegram**: у вас він, найімовірніше, вже є. Якщо ні: [telegram.org](https://telegram.org).

Акаунти Tailscale, Vercel і Supabase створимо в модулях 02 і 11, коли вони знадобляться.

## Перевірка

Ви можете залогінитись у кожен із семи сервісів. Підписки ChatGPT Plus і Claude Pro активні (видно у налаштуваннях акаунта).

## Куди зберегти доступи

Поки що збережіть паролі від усіх семи акаунтів у 1Password (стандартний сейф Personal). Окремий сейф для агента створимо в модулі 03.
