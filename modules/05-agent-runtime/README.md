# 05. Агент

Серце системи. Claude Code вже стоїть із модуля 01; тут додаємо Codex CLI (працює за підпискою ChatGPT), робочу теку агента і його "посадову інструкцію": файл, де описано, хто ви, що для вас важливо і як агент має поводитись.

## Що ви отримаєте

Повноцінний робочий простір агента на сервері: дві AI-системи за вашими підписками, структура тек для проєктів і пам'яті, персональний профіль під вас.

## Що зробити руками

1. Зайдіть на сервер: `ssh agent@agent-1`.
2. Встановіть і залогіньте Codex CLI:

   ```sh
   npm install -g @openai/codex
   codex
   ```

   Оберіть Sign in with ChatGPT і пройдіть вхід за посиланням.

3. Перевірте, що Claude Code теж залогінений під користувачем agent: команда `claude` відкриває сесію без запиту логіна. Якщо просить логін, пройдіть його (Sign in via subscription).

## Що зробить агент

Запустіть `claude` і вставте цілком файл
[prompts/agent-workspace.md](prompts/agent-workspace.md). Агент створить структуру тек і разом з вами заповнить ваш профіль: поставить питання про роль, компанію, пріоритети і збере відповіді у CLAUDE.md.

Відповідайте на питання профілю чесно і конкретно: що саме вважати терміновим, кого з людей ніколи не пропускати, якою мовою відповідати. Що точніший профіль, то корисніший агент.

## Перевірка

```sh
ls ~/workspace
```

Показує: `CLAUDE.md`, `projects/`, `memory/`, `inbox/`. Відкрийте CLAUDE.md і перечитайте профіль: там усе правда про вас.

## Куди зберегти доступи

Нових секретів у цьому модулі немає. Підписки Claude і ChatGPT авторизовані на пристрої, токени зберігаються локально самими CLI.
