# 12. Агенти і скіли

Розширюємо агента готовим набором навичок з відкритого репозиторію [claude-kit](https://github.com/borntojesus/claude-kit): десятки спеціалізованих агентів (маркетинг, продукт, аналітика, дослідження) і скілів, які підключаються до Claude Code однією командою.

## Що ви отримаєте

Агент вміє більше: глибокі дослідження з перевіркою джерел, робота з нотатками Obsidian, підготовка документів і презентацій, продуктові й маркетингові фреймворки. Все це доступно і з Telegram: просто просите, агент сам обирає потрібний скіл.

## Що зробити руками

Нічого, окрім двох команд. На сервері (`ssh agent@agent-1`) запустіть `claude` і виконайте по черзі:

```
/plugin marketplace add borntojesus/claude-kit
/plugin install claude-kit@claude-kit
```

Перезапустіть Claude Code після встановлення.

## Що зробить агент

Окремого промпта немає. Після встановлення напишіть агенту в Telegram: "переглянь встановлені скіли і скажи, які з них найкорисніші для мене з огляду на мій профіль". Агент прогляне набір і запропонує, що вимкнути як зайве.

## Перевірка

У Claude Code команда `/plugin` показує встановлений claude-kit. Спитайте агента в Telegram: "зроби глибоке дослідження ринку X" і подивіться, як він застосовує скіл дослідження.
