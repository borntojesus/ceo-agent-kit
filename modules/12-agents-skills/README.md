# 12. Агенти і скіли

Розширюємо агента готовим набором навичок з відкритого репозиторію [claude-kit](https://github.com/borntojesus/claude-kit): 100 спеціалізованих агентів і 64 скіли. Нижче готові набори за ролями: що вмикати CEO, фінансисту, маркетологу, продажнику й операційнику.

## Що зробити руками

На сервері (`ssh agent@agent-1`) запустіть `claude` і виконайте по черзі:

```
/plugin marketplace add borntojesus/claude-kit
/plugin install claude-kit@claude-kit
```

Перезапустіть Claude Code після встановлення.

## Що зробить агент

Напишіть агенту в Telegram: «активуй набір CEO» (або іншу роль з таблиць нижче). Він запише пріоритетний список у свій профіль і надалі сам братиме потрібного спеціаліста під задачу.

## CEO / фаундер

Дослідження (PwC, McKinsey, HBR 2025-2026) показують: найцінніше делегування для першої особи це підготовка до зустрічей, тріаж пошти, брифи для ради і ресерч. Під це:

| Що                             | Для чого                                            |
| ------------------------------ | --------------------------------------------------- |
| product-trend-researcher       | тренди і вікна можливостей раніше за конкурентів    |
| marketing-book-co-author       | ваші думки у thought leadership                     |
| product-manager (агент + скіл) | PRD, пріоритезація, роадмапи                        |
| zk-steward                     | звʼязана база знань фаундера                        |
| agents-orchestrator            | один агент, що диригує рештою                       |
| cowork (скіл)                  | делегування задач браузерним агентам одним промптом |

Правило ролі (Gartner): агент готує проводки і прогнози, людина їх затверджує. Жодних автономних платежів: це вшито і в правила кіта.

## CMO / маркетинг

Найсильніша вертикаль набору: понад 30 маркетинг-агентів і SEO-скілів.

| Що                                                                    | Для чого                                       |
| --------------------------------------------------------------------- | ---------------------------------------------- |
| marketing-content-creator                                             | контент-календарі і тексти під кожну платформу |
| marketing-seo-specialist + seo-audit (скіл)                           | органічний трафік: стратегія і повний аудит    |
| marketing-ai-citation-strategist + geo-content-optimizer (скіл)       | щоб вас цитували ChatGPT і Perplexity          |
| marketing-linkedin-content-creator                                    | LinkedIn-бренд фаундера                        |
| paid-media-auditor                                                    | аудит рекламних кабінетів на 200+ пунктів      |
| design-brand-guardian                                                 | консистентність бренду в усьому                |
| keyword-research + competitor-analysis + content-gap-analysis (скіли) | що писати і де конкуренти вас обганяють        |

Застереження ролі: кожне публічне твердження перевіряє людина. Вигадані факти в рекламі це вже штрафи регуляторів, а не теорія.

| Що                        | Для чого                                         |
| ------------------------- | ------------------------------------------------ |
| sales-outbound-strategist | ICP і багатоканальні секвенції                   |

| sales-pipeline-analyst    | здоровʼя пайплайна, застряглі угоди, прогноз     |

## HR / люди

Чесно: окремої HR-вертикалі в наборі поки немає. Що реально допомагає вже зараз:

| Що                                           | Для чого                                   |
| -------------------------------------------- | ------------------------------------------ |
| sales-coach                                  | коучингові фреймворки для розвитку команди |
| product-behavioral-nudge-engine              | поведінкова психологія мотивації і звичок  |
| specialized-cultural-intelligence-strategist | інклюзивна комунікація в команді           |
| humanizer (скіл)                             | внутрішні оголошення людською мовою        |

Тверде правило ролі: агент ніколи не ухвалює кадрових рішень (відмова, офер, оцінка). У EU з серпня 2026 AI у наймі це high-risk категорія за AI Act з обовʼязковим людським наглядом.
