# 01. Сервер Hetzner + Proxmox

Створюємо сервер, на якому житиме агент. Ставимо на нього Proxmox: систему керування контейнерами. Навіщо: агент живе у своєму контейнері, а кожен додатковий сервіс (власний пошук SearXNG, майбутні бази, дашборди) отримує свій окремий контейнер. Сервіси не заважають один одному, а зламаний експеримент зноситься одним кліком без шкоди агенту.

## Що ви отримаєте

- Сервер CX33 у Hetzner (близько 8 евро на місяць) з Proxmox.
- Контейнер `agent`: захищений Ubuntu-контейнер, у якому житиме агент.
- Місце під додаткові сервіси: кожен наступний піднімається окремим контейнером за хвилини.

Якщо сервісів стане багато, сервер апгрейдиться до CX42 (16 GB) кількома кліками в консолі Hetzner без переустановки.

## Що зробити руками

1. Згенеруйте SSH-ключ на своєму компʼютері, якщо його ще немає. Відкрийте Термінал (Mac: Terminal, Windows: PowerShell) і виконайте:

   ```sh
   ssh-keygen -t ed25519
   ```

   Тисніть Enter на всі питання. Ключ з'явиться у файлі `~/.ssh/id_ed25519.pub`.

2. У [console.hetzner.com](https://console.hetzner.com): New project → назвіть його `agent`.
3. Усередині проєкту: Add server.
   - Location: Nuremberg або Helsinki.
   - Image: **Debian 12** (на нього ставиться Proxmox).
   - Type: Shared vCPU → CX33 (4 vCPU, 8 GB RAM).
   - SSH keys → Add SSH key: вставте вміст файлу `~/.ssh/id_ed25519.pub` (виведіть його командою `cat ~/.ssh/id_ed25519.pub`).
   - Name: `agent-host`.
4. Натисніть Create & Buy now. Скопіюйте IP-адресу сервера.
5. Зайдіть на сервер із термінала і встановіть Claude Code:

   ```sh
   ssh root@ВАШ_IP
   curl -fsSL https://claude.ai/install.sh | bash
   claude
   ```

   Claude Code відкриє посилання для входу. Підтвердьте вхід через акаунт Claude.

## Що зробить агент

У запущеному Claude Code вставте цілком файл
[prompts/proxmox-install.md](prompts/proxmox-install.md). Агент захистить хост, встановить Proxmox, створить внутрішню мережу і контейнер `agent` з усім потрібним. Наприкінці покаже звіт.

## Додаткові сервіси

Коли база готова (після модуля 02), кожен новий сервіс це один промпт. Перший у черзі: власний пошуковик SearXNG для агента, промпт [searxng-lxc.md](../../prompts/searxng-lxc.md) у кореневій теці промптів.

## Перевірка

Після завершення промпта на сервері:

```sh
pct list
```

Показує контейнер `100 agent` у статусі running. Вхід у контейнер: `pct enter 100`.

## Куди зберегти доступи

- "Hetzner root password (agent-host)": пароль root із листа Hetzner, якщо приходив.
- "Proxmox root password": створить агент і покладе через op у модулі 03; до того тримайте у нотатках 1Password.
- IP сервера додайте в нотатку айтема.
