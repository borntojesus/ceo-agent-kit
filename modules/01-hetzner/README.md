# 01. Сервер Hetzner

Створюємо VPS, на якому житиме агент. Один сервер CX33, близько 8 евро на місяць, Німеччина або Фінляндія.

## Що ви отримаєте

Власний сервер Ubuntu з базовим захистом: окремий користувач для агента, файрвол, автоматичні оновлення безпеки, захист від перебору паролів.

## Що зробити руками

1. Згенеруйте SSH-ключ на своєму комп'ютері, якщо його ще немає. Відкрийте Термінал (Mac: Terminal, Windows: PowerShell) і виконайте:

   ```sh
   ssh-keygen -t ed25519
   ```

   Тисніть Enter на всі питання. Ключ з'явиться у файлі `~/.ssh/id_ed25519.pub`.

2. У [console.hetzner.com](https://console.hetzner.com): New project → назвіть його `agent`.
3. Усередині проєкту: Add server.
   - Location: Nuremberg або Helsinki.
   - Image: Ubuntu 24.04.
   - Type: Shared vCPU → CX33 (4 vCPU, 8 GB RAM). Цього достатньо.
   - SSH keys → Add SSH key: вставте вміст файлу `~/.ssh/id_ed25519.pub` (виведіть його командою `cat ~/.ssh/id_ed25519.pub`).
   - Name: `agent-1`.
4. Натисніть Create & Buy now. Скопіюйте IP-адресу сервера.
5. Зайдіть на сервер із термінала:

   ```sh
   ssh root@ВАШ_IP
   ```

6. Встановіть Claude Code прямо на сервері і залогіньтесь підпискою:

   ```sh
   curl -fsSL https://claude.ai/install.sh | bash
   claude
   ```

   Claude Code відкриє посилання для входу. Відкрийте його на комп'ютері, підтвердьте вхід через акаунт Claude.

## Що зробить агент

У запущеному Claude Code вставте цілком файл
[prompts/harden-vps.md](prompts/harden-vps.md). Агент створить користувача `agent`, увімкне файрвол, автооновлення і fail2ban, а наприкінці покаже звіт.

## Перевірка

Після завершення промпта вийдіть із сервера (`exit`) і зайдіть уже новим користувачем:

```sh
ssh agent@ВАШ_IP
```

Вхід працює без пароля, по ключу.

## Куди зберегти доступи

- "Hetzner root password (agent-1)": пароль root із листа Hetzner, якщо приходив.
- IP сервера додайте в нотатку айтема.
