# 09. Zed

Легкий модуль без промптів. Zed це швидкий редактор коду, через який зручно зрідка заглядати на сервер: подивитись файли агента, поправити свій профіль у CLAUDE.md, глянути журнали. Для щоденної роботи він не потрібен: усе робиться через Telegram.

## Що ви отримаєте

Редактор на вашому комп'ютері, який відкриває файли прямо на сервері через захищене з'єднання у вашій приватній мережі.

## Що зробити руками

1. Встановіть Zed: [zed.dev/download](https://zed.dev/download) (Mac, Windows або Linux).
2. Відкрийте Zed → у меню оберіть Remote Projects (Cmd+Shift+P → "remote") → Connect New Server.
3. Вкажіть адресу: `agent@agent-1` (працює, бо ваш комп'ютер у tailnet з модуля 02).
4. Відкрийте теку `/home/agent/workspace`.

## Перевірка

У Zed видно файли сервера: CLAUDE.md, projects, memory, inbox. Відкрийте CLAUDE.md, додайте пробіл, збережіть: файл зберігся на сервері.

## Куди зберегти доступи

Нових доступів немає: Zed використовує той самий SSH-ключ, що й термінал.
