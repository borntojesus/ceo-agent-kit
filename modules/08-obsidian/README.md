# 08. Obsidian

Ваш другий мозок. Нотатки живуть у вас на комп'ютері й телефоні в Obsidian, синхронізуються через приватне GitHub-репо, і агент бачить їх на сервері: читає, доповнює, веде щоденні записи.

## Що ви отримаєте

- Сховище нотаток Brain з простою структурою: швидкі записи, щоденні нотатки, проєкти, довідкові матеріали.
- Все, що ви кинете у нотатку "0-inbox", агент розбере і розкладе.
- Ранковий дайджест агент дописує у щоденну нотатку: історія ваших днів збирається сама.

## Що зробити руками

1. Встановіть [Obsidian](https://obsidian.md/download) на комп'ютер (і телефон за бажанням).
2. Створіть нове сховище (vault) з назвою `Brain`.
3. Створіть приватне репо на GitHub: [github.com/new](https://github.com/new) → назва `brain` → Private → Create.
4. В Obsidian встановіть плагін синхронізації: Settings → Community plugins → Turn on → Browse → знайдіть `Git` (obsidian-git) → Install → Enable.
5. Згенеруйте GitHub-токен для плагіна: [github.com/settings/personal-access-tokens](https://github.com/settings/personal-access-tokens) → Fine-grained token → доступ тільки до репо `brain`, права Contents: Read and write. Збережіть токен у 1Password.
6. Підключіть сховище до репо за інструкцією плагіна obsidian-git (Clone existing remote repo, вставте URL репо з токеном).

## Що зробить агент

На сервері запустіть `claude` і вставте цілком файл
[prompts/vault-sync.md](prompts/vault-sync.md). Агент склонує сховище на сервер, налаштує синхронізацію кожні 15 хвилин і створить стартову структуру нотаток.

## Перевірка

1. Створіть в Obsidian нотатку з текстом "тест синхронізації" і збережіть (плагін git запушить протягом кількох хвилин, або натисніть Commit and push вручну).
2. Напишіть агенту в Telegram: "що написано в моїй останній нотатці?". Він відповість текстом нотатки.

## Куди зберегти доступи

- "GitHub token (brain repo)": fine-grained токен з кроку 5, покладіть руками.
