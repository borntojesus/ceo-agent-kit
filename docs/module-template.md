# Шаблон модуля

Кожен модуль у `modules/NN-slug/README.md` пишеться українською за цією структурою. Промпти лежать поруч у `prompts/` і пишуться англійською.

Правила тексту: без тире як розділового знаку (кома, двокрапка, дужки або два речення), короткі речення, жодних згадок чужої інфраструктури.

# NN. Назва модуля

Один абзац: що це і навіщо саме вам.

## Що ви отримаєте

Список результатів простими словами. Що зміниться після проходження модуля.

## Що зробити руками

Нумерований список. Тільки дії, які без людини неможливі: оплата, реєстрація, OAuth-підтвердження, підтвердження телефоном. Кожен крок з точною назвою кнопки або сторінки.

1. Крок один.
2. Крок два.

## Що зробить агент

Відкрийте Claude Code на сервері і вставте цілком файл
[prompts/example-prompt.md](prompts/example-prompt.md). Дочекайтесь звіту агента.

Якщо промптів декілька, для кожного один рядок: коли його запускати і що він зробить.

## Перевірка

Як переконатися, що все працює. Конкретні команди або дії з очікуваним результатом.

## Куди зберегти доступи

Точні назви айтемів для 1Password (сейф Agent). Наприклад: "Telegram Bot Token (agent)".
