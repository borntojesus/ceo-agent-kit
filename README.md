# CEO Agent Kit

Приватний AI-агент для CEO, підприємців та C-level керівників. Ви розгортаєте його на власному сервері, на власних акаунтах і за власними підписками. До вашої пошти й нотаток має доступ тільки ваш сервер, і більше ніхто.

Онлайн-чеклист: [ai-kit.antonyuk.org](https://ai-kit.antonyuk.org)

## Що ви отримаєте

Особистого агента-асистента, який живе на вашому VPS і доступний тільки вам:

- розбирає дві ваші поштові скриньки, сортує листи по папках і підсвічує важливі
- пише вам у Telegram: окремі топіки для вхідних, пошти, проєктів і звітів
- щоранку надсилає дайджест, щоп'ятниці тижневий звіт
- веде ваш другий мозок в Obsidian: нотатки, проєкти, щоденні записи
- допомагає вайбкодити власні міні-проєкти на Vercel і Supabase
- жодного публічного порту: доступ тільки через ваш Tailscale VPN

## Головний принцип

Руками ви робите тільки те, що без вас неможливо: оплати, реєстрації, OAuth-підтвердження. Все інше налаштовує Claude Code. Кожен модуль містить готовий промпт: відкриваєте Claude Code на сервері, вставляєте промпт цілком, агент сам все встановлює і звітує.
